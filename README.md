Este manual está diseñado para:  

- 🧑‍🎓 **Estudiantes y académicos** que desean comprender los conceptos básicos de IA.  
- 💼 **Profesionales y expertos** que buscan referencias rápidas sobre aplicaciones prácticas de IA.
